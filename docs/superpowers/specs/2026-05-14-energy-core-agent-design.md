# Design Spec: Energy-centric Multi-agent Framework for Hierarchical Retrosynthesis (2026-05-14)

## Overview
This document outlines a high-level scientific slide deck design for the "Energy Core Agent" (ECA) project, a multi-agent framework designed to improve retrosynthesis planning using global thermodynamic consistency.

## Storyline
Current retrosynthesis methods suffer from thermodynamic invalidity, poor generalization to novel reactions, and search space divergence. ECA uses BDE/BDFE as a unified "energy language" to navigate the chemical space. The framework shifts the paradigm from pure pattern matching to physics-informed reasoning by enforcing global thermodynamic consistency checks throughout multi-step retrosynthesis.

## Core Contributions
1. Unified energy semantics using BDE/BDFE.
2. Global thermodynamic navigation mechanism.
3. Decoupled hierarchical agent architecture (single-step + multi-step).
4. Proven efficiency and robustness in retrosynthesis planning.
